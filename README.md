# Baqty Saver

An improved version of **Baqty Saver**, a browser extension designed to reduce internet data usage while browsing.

  <img src="https://raw.githubusercontent.com/OmarReed/baqty-saver-extension/refs/heads/main/logo.png" alt="Baqty Saver Screenshot" width="700">
</p>

## What’s New?

- Fixed the image-blocking behavior; large overlay layers no longer cover the page.
- Large images are replaced with a small card in the same image position that says: **"Click to show image"**.
- Images can be shown by clicking the card or hovering over it.
- Small images such as icons and avatars do not display a text card, so website layouts are not broken.
- Small images can still be shown by clicking or hovering over their area.
- Background protection continues to work through:
  - Service Worker
  - Alarms
  - Content Script inside pages
- Continued support for:
  - YouTube Controls
  - Facebook Reels / Blob Guard
